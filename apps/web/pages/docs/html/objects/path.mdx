## Introduction to paths

We've looked at simple shapes, then images. What about even more complex, rich shapes and content?
Meet the power couple â€” Path and Groups.

### Ading normal shape

```html title="index.html" output=true
<canvas
  id="c"
  width="200"
  style="border: 1px solid black"
  height="200"
></canvas>
<script src="https://cdn.jsdelivr.net/npm/fabric@latest/dist/fabric.js"></script>
<script>
  var canvas = new fabric.Canvas("c");
  var path = new fabric.Path("M 0 0 L 200 100 L 170 200 z");
  path.set({ left: 120, top: 120 });
  canvas.add(path);
</script>
```

---

Adding border and color

```html title="index.html" output=true
<canvas
  id="c"
  width="200"
  style="border: 1px solid black"
  height="200"
></canvas>
<script src="https://cdn.jsdelivr.net/npm/fabric@latest/dist/fabric.js"></script>
<script>
  var canvas = new fabric.Canvas("c");

  var path = new fabric.Path("M 0 0 L 100 100 L 200 100 z");
  path.set({ fill: "red", stroke: "green", opacity: 0.5 });
  canvas.add(path);
</script>
```
