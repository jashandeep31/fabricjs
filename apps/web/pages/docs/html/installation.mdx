Adding fabric.js to any html file is as simple as adding other packages to it. Just follow the guide and your first program using fabric.js will ready soon.

```html
<script src=https://cdn.jsdelivr.net/npm/fabric@latest/dist/index.js></script>
```

or

```bash
npm i fabric
```

<br />
## Hello world program

Now as we had installed our fabric js lets write are first **Hello world** program using the fabric js text [objects](/docs/html/objects/intro).

```html title="index.html" output=true {9-18}
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Hello to Fabric.JS</title>
  </head>
  <body>
    <canvas id="c" width="500" style="border: 1px solid black;" height="200"></canvas>
    <script src=https://cdn.jsdelivr.net/npm/fabric@latest/dist/index.js></script>
    <script>
      const canvas = new fabric.Canvas("c");
      const text = new fabric.Text("Hello Fabric.js", {
        fontSize: 20,
        fill: "green",
      });
      canvas.add(text);
    </script>
  </body>
</html>
```
